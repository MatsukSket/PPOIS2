# Shopping Mall Simulation

Это консольное приложение на Python, симулирующее работу торгового центра. 

---

## Основной функционал

* **Управление торговым центром:** Добавление новых магазинов (аренда мест в галерее) с учетом лимита вместимости.
* **Управление инвентарем:** Добавление товаров в магазины, отслеживание остатков (stock).
* **Система покупок:** Оформление заказов через кассовый аппарат, проверка баланса покупателя и наличия товара.
* **Скидки и акции:** Возможность активировать акции в отдельных магазинах и применять их для покупателей, участвующих в программе лояльности.
* **Поиск товаров:** Глобальный поиск по всем магазинам торгового центра по названию продукта.
* **Рейтинговая система:** Покупатели могут оценивать уровень обслуживания продавцов (от 1.0 до 5.0).
* **История покупок:** Просмотр купленных товаров в профиле покупателя.
